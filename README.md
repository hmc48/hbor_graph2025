# 全国病院指標 可視化ツール

厚生労働省のオープンデータを利用し、日本全国の病院指標を
グラフで可視化するサイトです。

病床数・平均在院日数・患者数などのデータから、
医療機関の特徴や傾向を確認できます。

医療関係者、医療政策、地域医療、医療データ分析などに
関心のある方に参考となる情報を公開しています。

---

## サイト

https://hmc48.github.io/hbor_graph2025/
---

## 主な機能

* 病院ごとの指標データの表示
* 病院間の比較グラフ
* 病床利用率などの指標表示
* 医療機関の特徴の可視化

---

## 使用データ

厚生労働省
医療機能情報提供制度（オープンデータ）

公開されている医療機関データをもとに作成しています。

---

## 注意事項

地域特性や診療機能の違いがあるため、
指標評価はあくまで参考値としてご利用ください。
